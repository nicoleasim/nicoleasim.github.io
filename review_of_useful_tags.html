Images

To insert an image, you will need to use the img tag. You will also need to ensure that the image you want to include is within your server and that you know the relative file location. In this example set of files the relative file location is "images/sleeping_cat_on_her_back.jpg". So we pop that into the image tag and voila!

We'll come back to images in week 7 :-) but don't forget to log any resources that you find and save into your metadata document. This adorable cat is from wikicommons, which is a great resource of copyright free/open usage material that you can use for your projects.

You can also set an img tag within the figure tag to let you add some mark-up to the image.

Audio

Audio content is nested in the audio tag. This tag also allows you to insert the audio controls needed to interact with the audio file. We will return to audio content in week 8! This audio file is available from the BBC Sound Effects website

Video

Video content uses the video tag. Much as with audio, this tag includes all the controls you need, and also has the option to attach closed captions/subtitles for accessibility purposes (we'll explore these more in Week 10!). There will be more on video content in week 9! This video file is available from pixabay and is free for usage without attribution.
